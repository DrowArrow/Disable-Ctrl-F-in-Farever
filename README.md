# Introduction
This AuotHotkey script is designed to disable "Ctrl + F" when Farever is the active window (allowing you to keep using ctrl + f in other applications without any issue while the script is running.) this specific script is also designed to check if the game process is running (Farever.exe) and if not it will tell steam to launch the game then wait for farever to be fully launched before moving on to disabling the hotkey when the game is active. on top of that, when the game closes the script is also designed to close it self so you don't have to remember to do it yourself.

## Running the script
To run the script you have two options:
1. Running the script directly from the .ahk file will require you to have autohotkey V2 installed
2. Downloading the exe which can be found in the releases, no extra steps are required as it packages everything it needs to run inside the .exe file.
